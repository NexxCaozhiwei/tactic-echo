Tactic Echo 0.9.50 — P5.4 分层冷却显示修复

本版解决“饰品能显示 CD，而大部分技能、药水没有 CD 时间”的问题。

- HUD 会直接显示 Blizzard 原生 `DurationObject` 的倒计时数字，覆盖直接动作条技能、SpellID 技能回退，以及药水/物品类别冷却。
- 直接默认动作条技能仍以动作条槽位为第一权威来源，因此 HUD 与动作条的数字保持同源。
- 参考 `CooldownTracking.lua` 增加事件驱动本地追踪：脱战缓存技能/充能数据，施法成功后启动兜底计时，并在 `0.01 / 0.05 / 0.15 / 0.35` 秒用技能 API 自动校正。
- 原生对象能够正常渲染时，本地追踪不会覆盖它；本地追踪只在原生冷却对象或公开数值不可用时兜底。
- 药水 `item_duration` 不再隐藏原生数字；饰品的槽位 API + ItemID 回退逻辑保持不变。
- 全部改动仅限 HUD 只读展示；官方推荐、动作派发、TEAP、TEK 和安全门禁未修改。

请将压缩包内容直接解压覆盖项目根目录；本包不包含新编译的 TEK.exe。

建议实机依次验证：
1. 复仇之怒、灰烬觉醒等长 CD 技能；
2. 打断、控制、防御及一个双充能技能；
3. 爆发药水；
4. 13 / 14 号饰品；
5. 技能在公共冷却期间、冷却缩减/重置后是否仍与动作条一致。
