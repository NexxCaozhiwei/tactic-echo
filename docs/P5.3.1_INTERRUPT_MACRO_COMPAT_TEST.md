# P5.3.1 自动打断与 legacy 宏验收

1. 覆盖 AddOn 后执行 `/reload` 与 `/tecache refresh`。
2. 使用严格 legacy 宏：

```lua
#showtooltip
/stopcasting
/cast [@focus,nodead] 反制射击
/cleartarget
/targetenemy
/cast 反制射击
/targetlasttarget
```

3. **当前目标读条、无敌对焦点**：在兼容证据稳定到 `3/3` 后，应出现 `compat_interrupt_candidate`，并可进入 `dispatchOrigin=reaction`。
4. **当前目标读条、存在敌对焦点**：不得派发 target 候选；应记录 `macro_managed_target_target_preempted_by_focus`。
5. **反制射击 CD 中**：应记录 `interrupt_action_cooldown`，主键回到官方正常序列。
6. **钢条**：不得出现 reaction candidate。
7. 执行 `/temapping p531_macro_compat`，确认动作条代表 SpellID 为 `147362` 且宏名“反射/打断/DDD”均可通过唯一语义关联；同名同技能两枚宏应保持拒绝。
