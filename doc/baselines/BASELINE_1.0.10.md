# Tactic Echo 1.0.10 基线

## 修订目标

收敛主推荐图标的运行状态呈现，不改变推荐、动作条绑定、TEAP、TEK 或 AutoBurst 输入链路。

1. 主键处于 `paused` 时，保留官方推荐图标、暂停色边框与“暂停”状态标签；移除灰度去色、半透明压暗和 `paused` 遮罩。
2. 主键正在施放当前推荐的普通技能时，状态标签必须显示“施法”。
3. 主键处于引导锁时，内部 `channeling_lock` 语义保持不变，但玩家可见状态标签统一显示“引导”，不再显示“引导锁”。
4. 暂停主键不得因为独立的可用性采样（资源、距离、目标、不可用）重新套用灰色去色；暂停是运行状态，不是技能不可用。

## 固定规则

- 状态优先级保持：异常/阻断 → 引导或蓄力锁 → 当前推荐技能施法 → 普通暂停 → 常规可用性/CD 展示。
- `paused` 的协议编码、TEK non-dispatch 门禁和 AutoBurst 行为完全不变。
- CD 转盘与 CD 数字仍可在暂停图标上显示；本轮移除的仅是运行状态带来的额外灰色遮罩与去色。
- 副卡（爆发、打断、防御等）维持现有暂停灰化语义；本轮只修改主键。

## 实机验收

- 手动暂停后，主推荐图标保持原始颜色和正常亮度，显示“暂停”，不得出现额外灰色压暗遮罩。
- 施放当前主推荐的普通读条技能时，主键显示“施法”。
- 引导当前技能或玩家处于引导锁时，主键显示“引导”；TEK 仍拒绝 `channeling` 帧。
- 暂停、施法、引导标签切换不得改变 BindingToken、TEAP state 或 AutoBurst 计划。
